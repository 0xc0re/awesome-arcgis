## Introduction

Work on top of:

* DB2
* Oracle
* PostgreSQL
* Microsoft SQL Server
* SAP HANA


It is multi user and editor, "unlimited" size.

Support [vector](../../../data-types/vector/README.md) and [raster](../../../data-types/raster/README.md) datasets
