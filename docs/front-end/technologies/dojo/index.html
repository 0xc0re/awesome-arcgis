Dojo

ArcGIS API for JavaScript uses internally Dojo Toolkit to implement more than 300 classes that allow you to build powerful maps, spatial analysis, data visualizations, etc.

Introduction

At this point in time there are two generations of the ArcGIS API been maintained:

  • 3.x generation: for 2D maps
  • 4.x generation: integrates 2D and 3D

If you are familiar with the ArcGIS API you can check this functionality matrix, if you are not you can ask for help on the JavaScript API group at GeoNet.
