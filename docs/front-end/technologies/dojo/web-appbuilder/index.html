Web AppBuilder for ArcGIS (Developer Edition)

Web AppBuilder is a WYSIWYG tool that allow any user to build map-centric app (with a full screen map) without writing any code. You as a web developer can add new widgets and themes or customize the existing ones.

Introduction

Web AppBuilder is:

  • For non-developers: A WYSIWYG tool that allow them to build and export responsive web apps adding and configuring pre-defined widgets and themes.

  • For developers: It is also a framework that can be extended adding custom themes and widgets.

The builder itself is written in NodeJS and can be downloaded from the developer console but you will need at least to have a free account in order to be able to download it.

In this page you will find some sites you can use to improve your skills to be able to extend Web AppBuilder.
